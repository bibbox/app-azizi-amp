The Azizi Anarchy Management Platform (Azizi AMP) is an incredibly powerful platform for downstream analysis and management of data collected with the versatile Open Data Kit (ODK) suite of tools. Azizi AMP has been developed purposefuly for the BiBBOX which is being developed by the B3Africa consortium.

Azizi AMP provides users the opportunity to:

  • Deploy mobile data collection tools with minimal IT support
  • Manage collected data easily
  • Extract the data to different formats
  • Integrate with other 3rd party applications eg. Baobab LIMS
  • Reduce the TAT from data collection to analysis

Azizi AMP is an essential tool for users in resource poor environment to employ cutting edge data collection and management practices without the heavy investments required in classic deployments
